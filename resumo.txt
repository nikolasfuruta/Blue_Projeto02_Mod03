Criar uma API backend onde existirão 3 rotas principais:


- Paises
     - nome
     - população
     - lingua mãe
     - PIB

- Estados
     - nome
     - região
     - população
     - valor salario minimo

- Cidades
     - nome
     - quantidade de bairros
     - população
     - data de aniversão da cidade




Dentro de cada rota precisamos ter as seguintes subrotas(sigam esses nomes por padrao):
- /add
- /update
- /delete
- /listall
- /listname
